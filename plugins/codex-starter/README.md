# Codex Starter Plugin

This folder is a publishable starter plugin scaffold for Codex.

## Included

- `.codex-plugin/plugin.json`
- `.mcp.json`
- `.app.json`
- `hooks.json`
- `skills/`
- `scripts/`
- `assets/`

## First edits to make

1. Rename the plugin if needed.
2. Replace URLs and metadata in `.codex-plugin/plugin.json`.
3. Add real assets or remove the placeholder image references.
4. Add your first skill under `skills/`.
